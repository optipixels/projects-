1. Getting Started


When you are ready to use this template, you must first upload the template files to a local or remote server. In this regard, unzip the template package you have downloaded from themeforest. In extracted folder you can see following folders:

Package contents

  • Template — Contains all the original template files comprises of HTML, CSS, JS etc. This is the folder which you'll upload on your server after necessary modifications.
  • Documentation — Inside this folder you will find this same documentation updated with the last changes.

Details regarding contents and structure of template folder is given below.

2. HTML Files


This template consist of 10 handcrafted HTML pages. All of these pages can be found at the root of the template folder. Despite the different contents of each page all of these pages have a common markup structure. Here is a very trimmed down version of a standard page structure. Note that for brevity, I'm not including content that isn’t relevant to the technique.

2.1. Skeleton Details


From the above skeleton it can be seen that there are 3 major logical code blocks in each page. Most of the contents displayed on the screen goes in one of this block.

header section contains the code for logo, primary navigation at top and of hero module. Except hero-module, content of this block mostly remains the same.

main section contains the page specific content which varies from page to page. For example, It contains different page sections, blog + portfolio + shop index enteries as well as their details.

footer section typically remains the same. It consist of social icons and copyright text note.

Further, skeleton shows two additional markup tags.sidebar contains all the widgets in sidebar. Whereas, section with the ID of content contains some elements with the sidebar Blog items are an example.

3. Stylesheet Files


All styling realted files of this template can be found in the folder named css.

css/
|
|-- style.css            # main styling file

|-- slickslider.css      # for slider styling

|-- jportilio.css        # portfolio items styling

4. Javascript Files


All client-side related javascript files can be found in the scripts folder.

scripts/
|
|--jQuery version 2.2.0 (jquery-2.2.0.min.js) 
|--Custom Scripts (scripts.js)
|--Count Down (jquery.countdown)
|--WOW plugin for animations (wow.min.js)
|--jQuery slider (jqueryslickslider.js)
|--jQuery Portfolio (jportilio.js)

In this folder there are 5 files. jquery.countdown.js, wow.min.js, jqueryslickslider.js, jportilio.js are third-party javascript libraries used in the template. These files are called after the footer tag of every page.

On the other hand, scripts.js file contains all the custom programmed javascript coded specifically for this template.
